Authoritative core

Measurement is the only stage treated as truth in canonical runtime storage and downstream grounding.

Best-effort symbolic

Symbolic notes are additive and useful, but not promoted to authoritative measurement.

Compatibility debt

The legacy flat phase1 path still exists and the UI still reconstructs a compatibility projection from canonical stages.
